# AirMind AI

An AI-Powered Urban Air Quality Intelligence Platform developed for the ET AI Hackathon.

## Problem Statement

Build an AI-powered platform that predicts air quality, identifies pollution hotspots, and provides actionable insights for smart city intervention.

## Features

- Live AQI Monitoring
- Weather Integration
- AQI Prediction using Machine Learning
- Interactive Pollution Maps
- Health Recommendations
- Analytics Dashboard
